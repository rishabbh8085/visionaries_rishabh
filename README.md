## 📌 Project Description

### 🔍 Problem
We are developing a full-Stack web application  to connect local sevice providers(plumbers, tutors, electricians, 
delivery agents) with customers.

### 💡 Solution
We provide a web interface through which the customer can connect to all the serive providers at one place.

### ✨ Key Features
- Feature 1: Easy booking system
- Feature 2: Provides Dashboard
- Feature 3: Rating and Review System
